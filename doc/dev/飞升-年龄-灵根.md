等级 = 获得等级（能量） = 获得等级（年龄*灵根）

飞升 = 能量 > 最大等级能量
    = 年龄*灵根 > 最大等级能量
    
设：每一个灵根是可以达到的等级(年龄),且年龄随等级增大
则：飞升 = x * 灵根 > 最大能量等级 （此时x小于最大年龄

即：人可以在最大年龄之前超过当前等级，但是又达不到下一个等级
